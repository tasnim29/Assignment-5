Question-3: What is event delegation in the DOM?

Event delegation in DOM is basically not giving the event listener to the child element, we will directly give the event listener to the parent node.By doing this,every child get access to the event listener

Question-4: What is event bubbling in the DOM?

There are two phases:1.Capture phase and 2.Bubbling phase.In capture phase it will go down from document to body to event.target.On the other hand,bubbling phase will bubble up to the top most element and get access of the target
